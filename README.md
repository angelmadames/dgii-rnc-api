## Overview

An Open-source Community-driven API for the National Taxpayer Registry (RNC)
of the Dominican Republic.

Created using [Nest](https://github.com/nestjs/nest).

## Installation

```bash
pnpm install
```

## Running the app

```bash
# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:production
```
